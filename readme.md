# Sow & Tell: A Social Network for Gardeners
## Final Project: Full-Stack Application Development


## Description

Gardening is a journey of patience, but the digital world is often too noisy for the quiet growth of a backyard allotment. Sow & Tell is a dedicated microblogging platform designed to bridge the gap between solitary gardening and community growth.

Our application provides a space where gardening enthusiasts can share "sprouts" (real-time updates), post seasonal harvest wins, and exchange hyper-local pest alerts. Unlike mainstream social media, Sow & Tell prioritizes climate-specific advice and authentic connection, celebrating the grit and reward of getting one's hands dirty.


## Technical Stack
 - Frontend: React.js, CSS3 (Flexbox/Grid), React Router
- Backend: Node.js, Express.js
- Database: MySQL via Sequelize ORM
- Authentication: JSON Web Tokens (JWT) & Bcrypt

## Project Structure & Features
- **User Authentication**: secure login and registration allowing gardeners to maintain personal profiles.
- **Microblogging Engine**: users can create, edit, and delete "sprout" posts categorized by plant type or garden task.
- **Responsive Design**: a mobile-friendly interface ensuring users can post alerts directly from their garden beds.
